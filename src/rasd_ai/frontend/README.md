# RASD Ops Center (Frontend)

## Setup

```powershell
npm install
```

## Run

```powershell
npm run dev
```

## Data files

Copy these files into `public/data/`:

- `nodes.json`
- `priorities.csv`
- `baseline_routes.json`
- `quantum_routes.json`
- `baseline_metrics_enriched.json`
- `quantum_metrics_enriched.json`
- `routes_for_frontend.json` (optional)

Logo:

- `public/logo/RASD.png`
